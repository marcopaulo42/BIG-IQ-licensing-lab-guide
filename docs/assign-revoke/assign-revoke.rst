Module 2: Assign/Revoke Licenses
********************************

BIG-IQ supports GUI- and API-driven methods for assigning and revoking licenses. Many internal and third-party
automation tools interface with BIG-IQ to offer VE licensing. This module will familiarize you with some of these
methods.
